    ABOUT

    What this version proves

    This is the first visual prototype for Develop Arkansas. It proves the user experience before expensive data engineering begins.

    1. A user can search and navigate a statewide map.
    2. Map layers can be turned on and off.
    3. A parcel can open a standardized property profile.
    4. The same interface can later connect to PostGIS, county GIS services, FEMA, utilities, and planning records.
